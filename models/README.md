# practicum_nn_task_01

### Папка для хранения обученных моделей.

Изначально тут сохранялась лучшая LSTM модель, обученная в 3 эпохи на словаре из 20 000 токенов. Файл **lstm_best.pt**

При переходе к сравнению с предобученной distilgpt2 сетью решил добавить вариан LSTM модели со словарём в 50 000 токенов и 10 эпох обучения.
Соответственно, такая сеть сохраняется в файл **lstm_50k_best.pt**.
